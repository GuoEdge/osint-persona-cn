# Web 控制台

本机网页界面，与 CLI 功能 1:1 对等。

## 安装与启动

### 一键启动（推荐，Windows）

双击项目根目录：

- **`start-osint-web.bat`**（或中文名 `启动情报台.bat`）— 启动服务并自动打开浏览器
- **`stop-osint-web.bat`**（或 `停止情报台.bat`）— 释放 8787 端口

固定到桌面（运行一次即可）：

```powershell
cd D:\gochj\gochj-web
powershell -ExecutionPolicy Bypass -File .\scripts\create-desktop-shortcut.ps1
```

会在桌面生成 **OSINT Web.lnk** 和 **Stop OSINT Web.lnk**。

脚本会自动使用 `.venv\Scripts\python.exe`（若存在），否则用系统 `python`，并设置 `PYTHONPATH=src`（无需每次手敲）。

### 命令行启动

```powershell
pip install -e ".[dev,web]"
osint web
```

浏览器打开：http://127.0.0.1:8787

可选参数：

```bash
osint web --port 8787 --host 127.0.0.1
```

## 搜罗模式说明

搜罗页工具栏中的 **「搜罗模式」**（原「配置」）用于快速切换信源方案。选择后：

1. 下方会显示该模式的 **说明卡片**（信源、适用场景、画像模拟默认值）  
2. 上方 **来源勾选** 会同步为该模式预设（仍可手动增删）  

| 模式 | 主要信源 | 说明 |
|------|----------|------|
| 默认 | 知乎、B站、网页、微信 | 日常搜罗；微信含阅读量质量过滤 |
| 全量 | + V2EX、RSS | 信源最全，耗时更长 |
| 深度研究 | 知乎、B站、V2EX、网页 | 不含微信；默认开启画像模拟 |
| 知乎深挖 | 仅知乎 | 问答/观点专题；默认跳过画像模拟 |

**能力说明**：各模式使用相同的 AI 与 pipeline 步骤；区别是 **并行采集哪些网站**，以及 **是否默认做画像兴趣模拟**。知乎深挖不会单独开启额外爬虫，但单源时知乎深度搜索（高赞回答展开等）效果更集中。

自定义模式见 `config.yaml` → `profiles:`，详见 [CAPABILITIES.md](CAPABILITIES.md#搜罗模式profiles)。

## 多任务搜罗与任务列表

- **开始搜罗**：聚焦当前任务，清空工作区并订阅进度（SSE）
- **加入队列**：后台排队，**不会**打断正在观看的搜罗进度
- **任务列表**（搜罗页折叠面板）：查看排队/进行中/近期完成；点击聚焦、可取消
- 默认最多 **2** 路同时运行（`search.max_concurrent_searches`），其余排队；排队上限默认 **20**

宽屏可勾选 **「结果/报告分屏」** 对照阅读；报告 Tab 或工具栏 **Focus** 适合长文沉浸阅读。

## 运行参数（设置页）

**设置 → 运行参数** 可在线调整（写入 `~/.osint/config.yaml` 并立即生效）：

| 分组 | 示例项 |
|------|--------|
| 搜罗并发 | 同时运行数、排队上限 |
| 搜罗加速 | 关联词发现、知乎深度采集、采集提前结束、AI 摘要条数 |
| 搜罗采集 | 扩展查询上限、评论挖掘条数 |
| 知乎开放平台 | 限流间隔、重试次数 |
| AI 行为 | 画像注入、自动重建 |

**说明**：已在进行中的搜罗不受本次并发修改影响；调高并发后排队任务会自动开始消化。
